Privacy Policy for G Task Express

G Task Express ("the extension") values your privacy. This Privacy Policy outlines the types of information we collect, how we use it, and the measures we take to protect your data.

Information We Access

The extension requires access to your Google Task lists to provide its functionality. Specifically, the extension accesses:

  • Your Google Task lists to display them in the extension popup and in the right-click menu.
  • The ability to create new Google Tasks on your behalf.

How We Use Your Information

The extension uses your Google Task data in the following ways:

  • Creating Tasks: Users can create tasks by entering a title, notes, and selecting a task list in the extension popup. The extension then uses the Google Tasks API to create a task on your behalf.
  • Refreshing Task Lists: Users can refresh the task lists in the popup to get the latest Google Task lists, allowing the extension to read the updated lists.
  • Right-Click Menu: Users can select text on a webpage and create a task using the selected text as the title or notes. Users can also create tasks using the webpage title as the task title and the URL as the notes through the right-click menu. The right-click menu also displays your Google Task lists, allowing you to select a task list when creating tasks from the menu. It also provides an option to refresh the Google Task lists.

Data Storage and Security

The extension does not store any user data outside the user’s local environment. OAuth tokens and any data accessed via the Google API are managed and stored by Google and your browser, not by the extension.

User Consent

The extension operates with your explicit consent when you grant it access to your Google Task lists through the OAuth process. You can revoke this access at any time through your Google account settings.

Compliance Disclosure

(G Task Express)’s use and transfer to any other app of information received from Google APIs will adhere to Google API Services User Data Policy, including the Limited Use requirements.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page, and we will notify users through the extension if significant changes are made.
